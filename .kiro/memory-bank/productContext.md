# Product Context

## Purpose
This project serves as a hands-on workshop for Kiro Dev, providing a structured environment to learn AI-assisted development workflows.

## Problems It Solves
- Provides a guided environment for learning Kiro Dev
- Demonstrates how to configure and use Memory Bank for persistent context
- Shows how to set up project rules for consistent AI assistance

## User Experience
- Workshop participants follow exercises to learn Kiro features
- The Memory Bank provides context continuity across sessions
- Rules in `.amazonq/rules/` guide Kiro behavior for the project
